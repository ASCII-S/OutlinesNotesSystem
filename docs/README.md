## 💡 使用建议

### 第一次使用

1. 按照 [快速开始](QUICKSTART.md) 完成初始化
2. 创建你的第一个主题和知识点
3. 运行 `./system/start.sh` 和 `./system/end.sh` 体验工作流
4. 阅读 [使用指南](USER_GUIDE.md) 了解更多功能

### 日常使用

1. 每天开始时运行 `./system/start.sh` 生成复习清单
2. 使用 Claude Code 或手动创建新的知识点
3. 完成复习后运行 `./system/end.sh` 同步进度
4. 定期查看统计报表了解学习进展

### 遇到问题时

1. 查看对应文档的"常见问题"章节
2. 检查 `config/kb_config.yaml` 配置是否正确
3. 运行 `./system/init.sh` 重新初始化
4. 查看 [开发指南](DEVELOPMENT.md) 了解项目结构
